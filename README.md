# 🌟 Project Overview

RechargePro is a Java-based web application designed to simulate a real-world mobile recharge platform.  
The system provides recharge management, transaction handling, validation mechanisms, and persistent recharge history storage through an interactive user interface.

# ⚙️ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Alvas-College-of-Engineering/Create-mobile-recharge-system.git
```

---

## 2️⃣ Open in Eclipse IDE

- Open Eclipse
- Import Existing Project
- Configure Apache Tomcat Server

---

## 3️⃣ Configure Database

Create MySQL Database:

```sql
CREATE DATABASE mobilerecharge;
```

Update JDBC credentials:

```java
String url = "jdbc:mysql://localhost:3306/mobilerecharge";
String username = "root";
String password = "your_password";
```

---

## 4️⃣ Run on Server

- Start Apache Tomcat
- Run the project

Open in browser:

```bash
http://localhost:8080/
```
